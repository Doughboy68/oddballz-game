ℹ️ ABOUT & CREDITS

Oddballz HD is inspired by a game designed by Fred Kohler in the early 1990s and based off the Windows 1992 port by Brian Semotiuk.

It uses the exact same algorithms and preserves authentic hexagonal strategy gameplay, enhanced with real-time 3D sphere physics, metallic shaders, dynamic space flight backdrops, and Web Audio synthesis.

Code and graphics/design copyright © Brian Semotiuk and Fred Kohler.
