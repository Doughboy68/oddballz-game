========================================================================
                      ODDBALLZ (1992 / Web Port v1.0)
          Created by Fred Kohler & Ported to Web by Brian Semotiuk
========================================================================

Oddballz is a classic 1992 hexagonal gravity puzzle game, fully ported to
run natively in any modern web browser (Chrome, Edge, Firefox, Safari)!

------------------------------------------------------------------------
HOW TO RUN:
------------------------------------------------------------------------
1. Extract the contents of this ZIP file to any folder.
2. Double-click 'index.html' to open and play the game directly in your browser.
3. No web server, node, or browser extensions needed!

------------------------------------------------------------------------
GAME CONTROLS:
------------------------------------------------------------------------
• Left / Right Arrow  : Move shape left / right
• Up Arrow           : Rotate shape counter-clockwise
• Down Arrow         : Rotate shape clockwise
• Home / 'X'         : Flip shape on X-axis
• End / 'Y'          : Flip shape on Y-axis
• Spacebar           : Zip drop shape to bottom (+1 Point bonus)
• 'F' or Insert      : Cycle colors (Color Match mode)
• Enter              : Start Game
• 'P'                : Pause / Resume
• 'R'                : Start 'Rows Game' mode
• 'C'                : Start 'Color Match' mode
• 'M'                : Toggle Audio / Music

------------------------------------------------------------------------
LEVEL & SHAPE PROGRESSION:
------------------------------------------------------------------------
Oddballz features 7 distinct shape types that unlock progressively as you level up:
• Level 1 - 2  : 2 Shape Types (3-4 Colors)
• Level 3 - 4  : 3 Shape Types (4-5 Colors)
• Level 5 - 6  : 4 Shape Types (5-6 Colors)
• Level 7 - 8  : 5 Shape Types (6 Colors)
• Level 9      : 6 Shape Types (6 Colors)
• Level 10 - 50: All 7 Shape Types Unlocked (6 Colors)

------------------------------------------------------------------------
FEATURES & HIGHLIGHTS:
------------------------------------------------------------------------
- Authentic 1992 320x240 VGA graphics with pixel-perfect ball masking
- Real-time in-game stat boxes in canvas corners (Level, Skill, Oddballz, Score)
- Both classic game modes: Color Match & Rows Game
- Full soundtrack audio (Intro, Gameplay, Game Over)
- High score leaderboard saved locally
- Touch controls for mobile & tablet support
